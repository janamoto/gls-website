# Marketing — Editorial (Ashley-inspired)

Dark editorial landing for GLS Infotech. Ultra-minimal canvas with oversized Fraunces display serif, Inter grotesk labels, thin hairline rules, marquee ticker, and indigo accents.

## Components
- `EditorialHeader.jsx` — fixed frosted top bar with nav.
- `EditorialHero.jsx` — viewport-filling 3-line serif headline.
- `Marquee.jsx` — horizontal infinite ticker.
- `Manifesto.jsx` — two-column approach statement.
- `ServicesIndex.jsx` — numbered services list with hover reveal.
- `ClientGrid.jsx` — 3×2 client wall.
- `ProcessList.jsx` — four-step vertical rhythm.
- `StatsBand.jsx` — four-column numerals band.
- `PullQuote.jsx` — centered italic testimonial.
- `ContactBlock.jsx` — oversized "Say hello" headline + contact grid.
- `EditorialFooter.jsx` — thin rule footer.

Custom cursor + grain overlay wired up in `index.html`.
